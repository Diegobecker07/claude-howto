# Gerador de Documentação de API

Gerar documentação de API completa:

1. Escanear endpoints de API
2. Extrair assinaturas de função e JSDoc
3. Organizar por módulo/endpoint
4. Criar markdown com exemplos
5. Incluir schemas de requisição/resposta
6. Adicionar documentação de erros
