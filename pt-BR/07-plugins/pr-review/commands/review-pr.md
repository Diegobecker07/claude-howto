# Revisão de PR

Este comando inicia uma revisão completa do pull request incluindo:

1. Análise de segurança
2. Verificação de cobertura de testes
3. Atualizações de documentação
4. Verificações de qualidade de código
5. Avaliação de impacto de desempenho
