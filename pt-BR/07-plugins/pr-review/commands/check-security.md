# Verificação de Segurança

Realizar análise de segurança focada nas mudanças de código:

1. Verificações de autenticação/autorização
2. Riscos de exposição de dados
3. Vulnerabilidades de injeção
4. Fraquezas criptográficas
5. Dados sensíveis em logs
