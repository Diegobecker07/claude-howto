# Verificação de Status do Sistema

Verificar a saúde do sistema em todos os serviços:

1. Consultar status dos pods do Kubernetes
2. Verificar conexões com o banco de dados
3. Monitorar tempos de resposta da API
4. Revisar taxas de erro
5. Verificar utilização de recursos
6. Reportar saúde geral
