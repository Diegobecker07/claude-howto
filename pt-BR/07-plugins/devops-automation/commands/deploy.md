# Deploy da Aplicação

Executar o fluxo de trabalho de deploy:

1. Executar verificações pré-deploy
2. Construir a aplicação
3. Executar testes
4. Fazer deploy no ambiente alvo
5. Executar verificações de saúde
6. Notificar a equipe no Slack
