## Visão geral

Este diretório concentra a localização pt-BR do guia, com foco em:

- manter comandos, caminhos, nomes de arquivos e blocos de código intactos
- traduzir apenas o texto explicativo para português natural e técnico
- preservar rastreabilidade de origem com metadados `i18n-*` no topo de cada arquivo
